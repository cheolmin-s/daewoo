포트폴리오 소개

제작기간 : 2주

이 포트폴리오는 미래에셋대우 사이트를 카피한 PC버전 포트폴리오 사이트입니다. 기존 사이트에서는 div를 이용한 마크업을 하였는데 시맨틱 태그를 사용하여 의미론적으로 다시 마크업을 했습니다.기본적으로 마크업 유효성 검사를 통해 웹 표준을 준수하였으며 접근성 또한 적절한 대체 텍스트와 모든 기능을 키보드로 사용이 가능하게 제작하였습니다.header 와 footer는 모든 페이지에 공통으로 들어가는 부분이라 따로 분리하여 include를 하였으며 각 페이지마다 css를 분리하여 좀 더 작업의 가독성을 높였습니다.
