자주하는 질문

  • Q ID, 이용자비밀번호, 공인인증서 비밀먼호를 잊어 버렸어요. 어떻게 해야 하나요? 내용보기
    A ID 찾기, 이용자비밀번호 재등록, 공인인증서 재발급 서비스를 이용하시면 됩니다.
    • ID찾기
      홈페이지 → 서비스신청/변경 → 고객 ID찾기
      ID찾기 바로가기
    • 이용자비밀번호 재등록
      홈페이지 → 서비스신청/변경 → 고객정보관리 → 이용자비밀번호재등록/변경
      이용자비밀번호 재등록/변경 바로가기
    • 공인인증서 재발급
      홈페이지 → 공인인증센터 → 인증서 발급/재발급
      공인인증센터 바로가기
  • Q 접속을 했는데 계좌번호가 보이지 않습니다. 어떻게 해야 하나요? 내용보기
    A ID별 계좌등록/해지메뉴에서 계좌가 미등록 된 경우 화면에서 계좌번호가 확인이 안됩니다.
    • ID별 계좌등록/해지
      홈페이지 → 서비스신청/변경 → 계좌개설 /ID 화면에서 등록하시면 가능합니다.
      ID별 계좌등록/해지 바로가기
    • 계좌를 오랫동안 사용하지 않을 경우 통폐합 또는 장기간 미사용계좌로 등록 될 수 있습니다.
      영업점 또는 고객센터에서 본인여부 확인 후 유선으로 해지 가능합니다.
    • 장기간 미사용계좌일 경우
      홈페이지 → 서비스신청/변경 → 고객정보관리 → 장기간 미사용 계좌 해제 화면에서 해제 가능합니다.
  • Q [계좌개설] 비대면앱으로 계좌개설을 했는데 이용절차는 어떻게 되나요? 내용보기
    A 비대면 계좌개설 완료 후에는 ID 등록이 완료된 상태이므로 인증서 발급 → HTS/모바일앱 설치 후
    이용하시면 됩니다.
    • 인증서 발급 바로가기 HTS 다운로드 바로가기
  • Q [퇴직연금] 아이디와 이용자비밀번호를 잊어버렸는데 어떻게 하나요? 내용보기
    A 홈페이지, 연금관리앱에서 확인이 가능합니다.
    • 계좌 보유 고객
      • 홈페이지 : 서비스신청/변경 → 계좌개설/ID → ID찾기
      • 홈페이지 : 서비스신청 → 고객정보관리 → 이용자비밀번호 재등록/변경
      • 연금관리앱 : 왼쪽 상단 전체메뉴 → 고객센타 → ID확인/비밀번호변경
    • 계좌 미보유 고객(퇴직연금제도만 가입되어있는 고객)
      • 아이디는 퇴직연금상담센터(02-3774-2000)로 연락주시면 본인 확인 후 SMS로 발송
      • 홈페이지 : 서비스신청/변경 → 고객정보관리 → 퇴직연금 이용자비밀번호 재등록
  • Q [스마트폰] HTS에서 조회했던 관심종목을 스마트폰으로도 조회할수 있나요? 내용보기
    A HTS에 등록한 관심종목을 서버에 저장하기를 하면 스마트폰에서도 조회할수 있습니다.
    • KAIROS : 관심종목화면 → 관심종목등록 → 관심종목 저장하기 → 서버에 저장하기
  • Q [ 계좌관리] 계좌번호를 모르는데 어떻게 해야 하나요? 내용보기
    A 가까운 미래에셋대우 영업점으로 방문하셔야 합니다.
    • 징규서류
      • 개인(본인) : 실명확인증표, 등록인감(서명)
      • 대리인 : 본인 실명확인증표, 등록인감(서명), 대리인실명확인증표, 인감증명서,
        위임장(인감증명서 인감날인)
